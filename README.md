# (GSI) Sistema de Reserva de Salas

Sistema de reserva de salas para professores do Instituto Politécnico Jean Piaget.
Desenvolvido no âmbito da Unidade Curricular de Gestão de Sistemas de Informação — 2025/2026.

##Descrição 🧠

Este mini-SI é uma aplicação web que permite aos professores consultar a disponibilidade de salas, efetuar reservas e gerir as suas ocupações em tempo real. O sistema centraliza a gestão de espaços académicos, eliminando conflitos de horário e reduzindo a necessidade de coordenação manual.

##Funcionalidades ⚙️

Dashboard — visão geral em tempo real com total de salas, salas livres, salas reservadas e percentagem de ocupação
Salas — listagem de todas as salas com tipo, capacidade e estado atual; reserva direta com validação de conflitos de horário
Reservas — consulta e cancelamento de reservas existentes
